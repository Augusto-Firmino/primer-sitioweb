Desarrollador Frontend

Frontend es la parte de un programa o dispositivo a la que un usuario puede acceder directamente. Son todas las tecnologías de diseño y desarrollo web que corren en el navegador y que se encargan de la interactividad con los usuarios. Un programador Frontend debe saber de código que entienda el navegador (HTML, CSS y JavaScript) para poder usar algunos frameworks o librerías que expanden sus capacidades para crear cualquier tipo de interfaces de usuarios. React, Redux, Angular, Bootstrap, Foundation, LESS, Sass, Stylus y PostCSS son algunos de ellos. Herramientas que maneja un frontend Debido a que un frontend es el desarrollador que va a manejar las cosas del lado del cliente, las tecnologías con las que va a trabajar son: HTML: Devedocs HTML CSS: Devedocs CSS JavaScript: Devedocs JavaScript Frameworks de CSS para frontend: Bootstrap: Getbootstrap Foundation CSS: Fundation CSS Materialize CSS: Meterialice CSS Los frameworks de JavaScript para frontend: React JS: React JS Angular JS: Angular JS Vue JS: Vue JS Preprocesadores de CSS: Stylus: Stylus SASS: SASS Compiladores / empaquetadores de JS: BABEL: Babel JS Webpack: Webpack

Desarrollador Backend

Backend en programación corresponde al lado opuesto a un Front-end en un sitio web o aplicación, ya que el Backend trabaja en el lado del servidor, mientras el Frontend lo hace en el lado del cliente. Es el responsable de manejar toda la lógica que hay detrás de una petición dada por el navegador hacia el servidor. Una característica que lo diferencia del Frontend es que no tiene estándares, puesto que tiene varios lenguajes de programación (Node.js, Python, PHP, Ruby, GO, Java, .NET entre otros) con los que debe trabajar. Cada uno de estos lenguajes tiene sus propios frameworks como: Django (Python) Lavarel (PHP) Rails (Ruby) Express (JavaScript) Spring (Java) El Backend también tiene en cuenta la infraestructura donde va a realizarse el deploy de su aplicación (esto también puede ser tarea de un DevOps, un perfil dedicado a la infraestructura), con tecnologías como: Google Cloud DigitalOcean AWS Heroku, entre otras. ¿Qué es deploy? Deploy es un término famoso entre los desarrolladores web. Puede significar muchas cosas, dependiendo del ambiente y de la tecnología usada.

Desarrollador FullStack

¿Qué es el desarrollador de pila completa? Es un experto con diversas habilidades informáticas. Es decir, es un perfil mixto multihabilidad que puede funcionar tanto en front end como en back end de desarrollo web. Está lleno de conocimientos y herramientas que le permiten abordar cualquier etapa de la construcción de un portal.

HTML

Una explicación informal de lo que es HTML HTML es la mejor forma tecnológica de almacenar y consultar información. Puede almacenar información escribiéndola en papel, esculpiéndola en piedra, grabándola en una cinta de video o utilizando un dispositivo de grabación. Sin embargo, HTML sigue siendo mejor que todos estos medios combinados. En HTML, puede insertar video, audio y texto del curso en un documento. Pero no solo eso, también es interactivo. Puede navegar por su contenido a través de los enlaces. Y no se detiene allí. Puede ver el HTML en su computadora, en su teléfono, en su reloj y, si tiene HTML moderno, quizás en su lavadora. Como puede ver, HTML es texto, video, audio y más. Por si fuera poco, HTML es el lenguaje que utilizan los sitios web. ¿Qué significa eso?. Ahí es cuando presiono guardar documento, en ese momento exacto. Cualquier persona en cualquier parte del mundo con conexión a Internet puede ver la información que ha registrado.

CSS

¿Qué es CSS? CSS significa hojas de estilo en cascada. Básicamente es un lenguaje que gestiona el diseño y la presentación de las páginas web, es decir, cómo se ven cuando los usuarios las visitan. Funciona con HTML que se encarga del contenido central de las páginas. Se denominan hojas de estilo "en cascada " porque puede tener varias hojas de las cuales una ha heredado (o "en cascada ") propiedades de las otras hojas. Para muchas personas, una simple plantilla de blog será suficiente.

JavaScript

JavaScript (o "JS ") es el lenguaje de programación más utilizado para la programación dinámica del lado del cliente en páginas web, pero también es muy popular en el lado del servidor (en EE. UU.), utilizando el entorno en el que se ejecuta el script como Node. .js.

JavaScript no debe confundirse con el lenguaje de programación Java. Aunque "Java " y "JavaScript " son marcas comerciales (o marcas comerciales registradas) de Oracle en los Estados Unidos y otros países, los dos lenguajes de programación son muy diferentes en términos de sintaxis, semántica y casos de aplicación. JavaScript se usa principalmente en los navegadores, lo que permite a los desarrolladores manipular el contenido de la página web a través del DOM, manipular datos usando AJAX e IndexedDB, dibujar gráficos usando la mesa de trabajo, interactuar con dispositivos que ejecutan el navegador a través de varias API, etc. JavaScript es uno de los lenguajes más utilizados en el mundo, debido al reciente desarrollo y mejora del rendimiento de las API disponibles en el navegador.
